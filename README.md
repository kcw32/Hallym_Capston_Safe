# 프로젝트 개요
요근래 성범죄가 엄청난 사회적 이슈가 되고있다. 직접적인 피해를 입는 성범죄도 있지만 많은 전자기기들이 대중화가 되고 난 후
타인을 몰래 찍는 몰래카메라 범죄가 나날이 기승을 부리고 있다. 그래서 이러한 몰래카메라 범죄를 예방을 하기위해 safe camera
프로젝트를 계획했다.
